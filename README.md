# QuickCommerce MIS Dashboard (Excel + Power Query)

## Project Overview
This project demonstrates an end-to-end automated MIS dashboard built using Microsoft Excel and Power Query.

## Objective
To transform raw order data into a clean master dataset and build interactive KPIs and MIS dashboards.

## Tools Used
- Microsoft Excel
- Power Query
- Pivot Tables
- Slicers

## Key Features
- Automated data transformation using Power Query
- Code-to-name mapping using reference tables
- SLA vs Actual delivery analysis
- Dynamic KPIs & MIS Dashboard

## Automation Flow
Raw Data → Power Query → Master Sheet → KPI & MIS Dashboard

## How to Use
1. Download the Excel file
2. Open the file
3. Data → Refresh All
4. Use slicers to explore insights

## Disclaimer
This is a self-learning project with simulated data for educational purposes.
